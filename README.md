# JavaScript Logic Programs

This project contains simple JavaScript programs to practice:

- Conditional statements (`if-else`)
- Switch statements
- Ternary operators
- Assignment operators
- Basic calculator logic

## Features

### Simple Calculator
Performs:
- Addition
- Subtraction
- Multiplication
- Division
- Invalid operator handling
- Division by zero handling

### Student Result System
- Adds bonus marks using `+=`
- Calculates grade based on marks
- Determines Pass/Fail using ternary operator
- Gives remarks using `switch`
- Handles invalid marks input

## Technologies Used

- JavaScript
- Console Output
